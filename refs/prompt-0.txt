Para o scraping, proponho o seguinte:

1. Criação de um segundo dataframe para as "home organizations".
2. Criar colunas para acompanhar e registrar cada etapa do processo:
- Se o website foi achado
- Qual website visitado
- Se a página foi raspada

Assim, conseguimos acompanhar os pontos de falha com maior detalhe.
A classificação precisa ser apenas "Yes" ou "No" e acontecer somente se todo o processo deu certo. Caso aconteça erro em etapa anterior, ou a classificação falhe, temos colunas específicas para isso -- aceito sugestões para deixar esse procedimento mais robusto.

Ao fim de tudo, adicionamos a coluna de is_Insurance na tabela de nomes, que é o resultado final.
